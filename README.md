
# Servidor de contenido

Servidor para carga y descarga de archivos, hecho en DJANGO.

## Documentation

Este simple proyecto permite desplegar un servidor de contenido para cargar y descargar archivos a través de HTTP o HTTPS. 

Los archivos cargados se alojarán por defecto en el directorio "media" del proyecto.
